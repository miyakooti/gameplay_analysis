ここからは2値型の特徴量を作成していきます。作成元となる対象はwhyplay, Gender, Work, Playstyle, Degreeです。¶

  • whyplay
    • ゲームをプレイする理由です。楽しむため、上達するため、勝利するためなどの項目がありますが、「楽しむためかどうか」という変数に落とし込んでいます。
  • Gender
    • maleとfemaleなので単にそれを区別するだけです。
  • Work
    • 雇用形態です。学生や無職、フリーターなどありますが、働いているかどうかが満足度に寄与すると考えたため、職を持っているかどうかを判定してカラムに格納しています。
  • PlayStype
    • オンラインやプレイ人数などのプレイスタイルです。複数人でやるかどうかが重要そうなのでそこを判定しています。
  • Degree
    • 学歴です。今回はMasterかBachelorをまとめて「大卒以上」とし、orで大卒であるかどうかを判定しています。

    pmatrix[i,]=(list(format(i, text)))#0の後に変数の数がくる(ex.4変数のとき04b)

for i in range (2**p-2):
    for j in range ((i+1),2**p-1):
        for k in range((j+1),2**p):    
            sum1=0
            for num in range (n):
                sum1=sum1+min(p-np.sum(pmatrix[i]==data[num]),p-np.sum(pmatrix[j]==data[num]),p-np.sum(pmatrix[k]==data[num]))

# 新しい特徴量の初期化
principal_features = np.zeros((len(df), 3), dtype=int)

# 各行に対して最も近いプリンシパルポイントを特定
for i, row in enumerate(df.iterrows()):
    distances = np.sum(np.abs(np.array(row[1][['for_fun', 'male', 'has_work', 'is_single_player', 'is_highly_educated']]) - np.array(principal_points)), axis=1)
    nearest_point_index = np.argmin(distances)
    principal_features[i, nearest_point_index] = 1

# 新しい特徴量をDataFrameに追加
df['principal_1'] = principal_features[:, 0]
df['principal_2'] = principal_features[:, 1]
df['principal_3'] = principal_features[:, 2]

pmatrix=np.empty((2**p,p))
for i in range(2**p):
    pmatrix[i,]=(list(format(i, '03b')))#0の後に変数の数がくる(ex.4変数のとき04b)


###STEP2 Prinsipal Pointsを探索###
#Step2　principal pointsを探索
sum2=1000
for i in range (2**p-2):
    for j in range ((i+1),2**p-1):
        for k in range((j+1),2**p):    
            sum1=0
            for num in range (n):
                sum1=sum1+min(p-np.sum(pmatrix[i]==data[num]),p-np.sum(pmatrix[j]==data[num]),p-np.sum(pmatrix[k]==data[num]))
            if sum1<=sum2:
                sum2=sum1
                pp1=i
                pp2=j
                pp3=k
                print('距離',sum2)
                print(list(format(pp1, '03b')) ,list(format(pp2, '03b')),list(format(pp3, '03b')))
